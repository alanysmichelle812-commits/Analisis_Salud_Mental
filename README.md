#  Análisis Exploratorio de Datos (EDA) de Tasas de Suicidio

Este repositorio contiene el Análisis Exploratorio de Datos (EDA) del dataset *crude suicide rates.csv* (Tasas Brutas de Suicidio), realizado con el objetivo de preparar variables para modelos de Regresión y Clasificación.

El análisis completo y los resultados detallados se encuentran en el archivo **`ANALYSIS.md`**.

---

## Cómo Ejecutar el Análisis

Para reproducir la limpieza de datos y el análisis exploratorio:

1.  **Instalar Dependencias:** Se requieren las librerías `pandas` y `numpy`. Instálalas en tu entorno de Python:
    ```bash
    pip install pandas numpy
    ```
2.  **Ejecutar el Script:** Abre la terminal en la carpeta del proyecto y ejecuta el script de Python:
    ```bash
    python eda_salud_mental.py
    ```

El script imprimirá la limpieza de datos y los resultados del EDA en la consola.
